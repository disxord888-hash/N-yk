# yk サイズ命名規則 仕様書
（20yk フォーマット命名体系）

## 1. 命名思想（由来）

yk は バイトサイズを 2 の累乗指数で表す命名単位である。

**表記形式：**
Nyk = 2^N bytes

数値は **指数そのもの** を表す。
SI（kB, MB）ではなく **2進基準** である。

**例：**
20yk = 2^20 bytes = 1,048,576 bytes (1MB)

## 2. フォーマット名変更

| 旧名称 | 新名称 |
| :--- | :--- |
| .11yk | .20yk |

**理由：**
- 制限値が 2^20B（1MB） のため。
- フォーマット名と物理制限を一致させるため。

## 4. フォーマット命名ルール（一般化）

この体系により、以下が成立する：

- **.20yk** → 最大 2²⁰B 制限フォーマット
- **.22yk** → 最大 2²²B 制限フォーマット
- **.34yk** → 最大 2³⁴B 制限フォーマット

👉 **拡張子そのものがサイズ仕様** となる。

## 5. 20yk フォーマットの正式定義（改名後）

- **フォーマット名**：.20yk
- **実体**：ZIP
- **制限**：単一ファイルの展開後サイズ ≤ 2²⁰B
- **制限違反時**：破損扱い
